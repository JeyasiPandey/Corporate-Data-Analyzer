# 📊 Corporate Data Analyzer

A Python-based desktop GUI application to analyze corporate datasets, generate reports, and visualize insights through interactive charts.

---

## 🚀 Features

* 📂 Import data from **CSV / Excel files**
* 🔍 Perform **GroupBy & Aggregations**:

  * Sum, Mean, Max, Min, Count, Median
* 📈 Generate **interactive charts**:

  * Bar Chart
  * Column Chart
  * Line Chart
  * Pie Chart
* 📤 Export reports:

  * Excel (.xlsx)
  * CSV (.csv)
* 🖼 Export charts as **PNG images**

---

## 🛠 Tech Stack

* **Python**
* **Tkinter** (GUI)
* **Pandas** (Data Processing)
* **NumPy**
* **Matplotlib** (Visualization)

---

## ▶️ How to Run

Make sure Python is installed on your system.

### 1️⃣ Install dependencies

```bash
pip install pandas numpy matplotlib
```

### 2️⃣ Run the application

```bash
python CorporateDataAnalyzer.py
```
